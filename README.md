# Hotel Booking Website

A full-stack hotel booking web application developed for HCL Hackathon.

## Features

- User Registration & Login
- Hotel Listing
- Room Booking
- Booking Confirmation
- Email Notifications
- Admin Room Management
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Spring Boot
- REST API

### Database
- MySQL
